# Healthcare Demand & Capacity Analysis

## Overview
This project analyses healthcare demand, capacity strain, and cost pressure using SQL Server.

## Key Features
- Rolling statistical baseline detection
- Demand overload detection
- Practice capacity analysis
- Advanced SQL window functions

## Architecture
ETL → Data Quality → Reporting → Analysis

## Insights Generated
- Identifies overloaded healthcare days
- Detects underutilisation
- Highlights high-pressure practices
- Supports operational decision-making

## Skills Demonstrated
- Advanced SQL (window functions)
- Data modelling
- Statistical analysis
- Business insight generation
