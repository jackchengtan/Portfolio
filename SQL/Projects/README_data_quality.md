# Healthcare Data Quality Monitoring Project

## Overview
This project simulates a real-world healthcare data quality monitoring system built using SQL Server.

## Features
- Daily metrics calculation
- Anomaly detection using rolling averages
- Data quality result tracking

## Tech Stack
- SQL Server
- Stored Procedures
- Window Functions

## How it works
1. ETL loads patient encounter data
2. Data quality procedures calculate metrics
3. System detects anomalies
4. Results stored for monitoring

## Skills Demonstrated
- SQL analytics
- Data quality frameworks
- Statistical anomaly detection
