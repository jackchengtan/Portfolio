# SQL Business Case Studies Portfolio

## Overview

This repository showcases a collection of SQL-based data analysis projects built using real-world business scenarios from the 8 Week SQL Challenge.

The goal of this portfolio is to demonstrate how SQL can be used to solve practical business problems across different domains, including customer analytics, operations, and delivery performance.

Each project focuses on transforming raw data into meaningful insights and actionable recommendations.

---

## Projects

### 1. Pizza Delivery Operations Analysis

**Focus:** Operations & logistics performance

* Cleaned messy operational data (null values, inconsistent formats)
* Analysed delivery success rates and runner performance
* Evaluated preparation time vs order size
* Calculated delivery speed and operational efficiency

**Key Skills:**

* Data cleaning
* Joins and aggregations
* Window functions
* Business metric calculation

---
